ESTATE & CO — A Modern Real Estate HTML Template

Estate & Co is a clean, fully responsive HTML/CSS/JS template built for 
real estate agencies, realtors, and property listing platforms. It comes 
with everything needed to launch a professional property website — no 
frameworks, no build tools, just open and edit.

✦ WHAT'S INCLUDED
- Home page with hero search bar, featured listings, stats, and testimonials
- About page with company story, values, and team section
- Contact page with working front-end form validation
- Property Listings page with sidebar filters (price, type, city, bedrooms) and pagination
- Property Detail page with image gallery, specs, amenities, agent card, and booking form
- Sign Up page with password match validation
- Sign In page with social login buttons (UI only)

✦ KEY FEATURES
- 100% responsive — looks great on desktop, tablet, and mobile
- Pure HTML5, CSS3, and vanilla JavaScript (no jQuery, no frameworks)
- Clean, well-commented, easy-to-edit code
- Custom typography pairing (Fraunces + Public Sans) for a premium feel
- Reusable property card and filter components
- Client-side form validation included
- Cross-browser compatible (Chrome, Firefox, Safari, Edge)
- Easy to connect to any backend (PHP, Node, Laravel, etc.)

✦ WHO IT'S FOR
- Real estate agencies and brokers
- Property listing marketplaces
- Freelancers/agencies building real estate sites for clients
- Developers who want a head start instead of building from scratch

✦ FILE STRUCTURE
index.html, about.html, contact.html, listings.html, listing-detail.html, 
signup.html, signin.html, /css/style.css, /js/main.js
